# 自选基金助手

Android 原生 APP，实时查看基金涨跌幅，管理自选基金组合。

## 主要功能

### 首页
- 自选基金列表，实时显示最新净值、涨跌幅、涨跌额
- 支持显示近一年收益率、持有额、估算收益、持有收益
- 特别关注基金
- 多维度排序（名称/涨跌幅/持有额/收益）
- 指数行情栏：18个指数支持点击查看实时详情

### 基金详情
- 估值走势：当日分时估值曲线
- 持仓明细：前十大重仓股及涨跌幅
- 历史净值：折线图/图表/列表三模式，六档时间切换
- 累计收益：收益率走势+基准对比，支持日期搜索
- 基金概况：基本信息、基金经理、历史业绩排名

### 行情中心
- 大盘资金流向（主力/超大/大/中/小单）
- 行业板块主力净流入排行+图表
- 北向/南向资金分时走势+汇总

### 其他
- 深色模式
- 持有份额与成本价管理
- 自选基金导入导出
